} } # Cómo publicarlo ## Opción A: README de perfil 1. Crea un repo llamado **i7protocol/i7protocol**. 2. Copia el archivo `README.md` de arriba como contenido del repo. Eso se verá en tu perfil. ## Opción B: Portfolio con GitHub Pages 1. Crea un repo llamado **i7protocol.github.io**. 2. Agrega los archivos `index.html` y `styles.css` tal cual. 3. Confirma que en `index.html` la constante `GH_USER` sea `"i7protocol"`. 4. Hace commit en `main` y activá GitHub Pages si no aparece solo. 5. Listo: tu portfolio estará online en `https://i7protocol.github.io`. ## Opción C: Portfolio dentro de un repo existente 1. En cualquier repo, crea una carpeta `/docs` y mete ahí `index.html` y `styles.css`. 2. En Settings → Pages, elegí **Deploy from a branch** y la carpeta `/docs`. 3. Espera el deploy. Fin. ## Personalización mínima - Cambiá la tagline en el header. - Editá la sección "Sobre mí". - Ajustá `MAX_REPOS` para mostrar más o menos. - Si querés fijar proyectos específicos arriba, cambiá el `sort` en `loadRepos` o crea una sección manual con links.
